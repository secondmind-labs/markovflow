:py:mod:`markovflow`
====================

.. py:module:: markovflow

.. autoapi-nested-parse::

   API reference documentation for the Markovflow package.

   Markovflow includes :mod:`ready-to-use GP models <markovflow.models>` for classification and
   regression.

   If you want to create your own model, you must implement either the
   :class:`~markovflow.models.models.MarkovFlowModel` or the
   :class:`~markovflow.models.models.MarkovFlowSparseModel` base class.
