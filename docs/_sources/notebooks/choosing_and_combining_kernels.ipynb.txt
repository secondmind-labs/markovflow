    "### Stationary kernels\n",
    "\n",
    "\n",
    "Matern kernels are stationary. A kernel is stationary if the joint distribution between $f(x)$ and $f(x^{\\prime})$ is the same as the joint distribution $f(x+Δ)$ and $f(x^{\\prime}+Δ)$. $x$ and $x^{\\prime}$ are some arbitrary locations. $Δ$ is an arbitrary real number. $f(x)$ is the random variable that represents the possible values of the underlying function at location $x$. $f(x^{\\prime})$ is the random variable that represents the possible values of the underlying function at location $x^{\\prime}$.\n",
    "\n",
    "An equivalent definition of a stationary kernel is: when the kernel is a function of $x-x^{\\prime}$, it is a stationary kernel.\n",
    "\n",
    "\n",
    "### Which Matern should I use?\n",
    "\n",
    "Pick a Matern kernel that is consistent with some property of the data that you have. If you are modelling stock prices, the price time series is not smooth, so a Matern1/2 kernel might be a better choice than a Matern5/2. On the other hand, if you are are modelling temperature during a day, which is a smooth series, a Matern3/2 or Matern5/2 kernel might be better than a Matern1/2 kernel.\n",
    "\n",
    "You can also draw samples from the corresponding prior with your best guesses for the hyperparameter values, and compare the look-and-feel of those samples with your data."
   ]

    "### Why samples from a GP with constant kernel are horizontal straight lines\n",
    "\n",
    "For a zero-meaned GP with a constant kernel $k(x, x^{\\prime})=\\sigma^2$, we can define the following equivalent process:\n",
    "\n",
    "$$f(x) = \\eta\n",
    "$$\n",
    "\n",
    "where $\\eta \\sim \\mathcal{N}(0, \\sigma^2)$.\n",
    "\n",
    "To verify that $f(x)$ defined this way has the same distribution as our zero-meaned GP with a constant kernel, we verify that:\n",
    "\n",
    "1. the mean of $f(x)$ for an arbitrary value of $x$ is 0.\n",
    "2. the covariance between $f(x)$ and $f(x^{\\prime})$ for arbitrary values of $x$ and $x^{\\prime}$, denoted by $cov(f(x), f(x^{\\prime}))$, equals $\\sigma^2$.\n",
    "\n",
    "Verify the mean:\n",
    "\n",
    "Because $f(x)=\\eta$, and $\\eta \\sim \\mathcal{N}(0, \\sigma^2)$, so $f(x)$ has zero mean.\n",
    "\n",
    "Verify the covariance:\n",
    "\n",
    "$\\begin{align}\n",
    "cov(f(x), f(x^{\\prime})) &= \\mathbb{E}\\left[(\\eta - 0)(\\eta - 0)\\right] \\\\\n",
    "&= cov(\\eta, \\eta) \\\\\n",
    "&= \\sigma^2 \\\\\n",
    "\\end{align}$\n",
    "\n",
    "Indeed $f(x)=\\eta$ is equivalent to our zero-meaned GP with a constant kernel. In other words, $f(x)=\\eta$ models the same set of functions as the GP does. From $f(x)=\\eta$, it is obvious that these functions are horizontal lines. And the model parameter $\\sigma^2$ decides the possible y-axis locations of those horizontal lines.\n",
    "\n",
    "\n",
    "\n",
    "### Why the constant kernel is useful\n",
    "\n",
    "From looking at the kernel samples figures, it is not obvious why the constant kernel is useful. It is better to understand its use case from the predictive distribution for test location $X_*$. Let's assume that there is no observation noise.\n",
    "\n",
    "The following equation shows the joint distribution between the latent random variable $f(X_*)$ and the observation random variable $y(X)$:\n",
    "\n",
    "$\\begin{bmatrix}f(X_*)\\\\y(X)\\end{bmatrix} \\sim \\mathcal{N}\\left(0,\\;\\; \\begin{bmatrix}k(X_*, X_*)&k(X_*, X)\\\\k(X_*,X)^{\\top}&k(X, X)\\end{bmatrix}\\right)$\n",
    "\n",
    "Apply the multivariate Gaussian conditional rule to derive the predictive distribution:\n",
    "\n",
    "$f(X_*)|y(X) \\sim \\mathcal{N}\\left(k(X_*, X)k(X,X)^{-1} y(X),\\;\\;k(X_*,X_*)-k(X_*,X)k(X,X)^{-1}k(X_*, X)^{\\top}\\right)$\n",
    "\n",
    "This predictive distribution has mean:\n",
    "\n",
    "$k(X_*, X)k(X,X)^{-1} y(X)$\n",
    "\n",
    "After we plug in the data $Y$, we get:\n",
    "\n",
    "$k(X_*, X)k(X,X)^{-1} Y$\n",
    "\n",
    "Imagine that the test location $X_*$ is far from the training location $X$. If we are using a Matern kernel, $k(X_*, X)$ will be 0, and therefore the predictive mean will be 0. However, if we are using a constant kernel, $k(X_*, X)=\\sigma^2$, no matter how far $X_*$ is from $X$. So the predictive mean will not be 0.\n",
    "\n",
    "This is the use case for the constant kernel -- to modify the mean of the predictive distribution. In practice, it is more common to use the constant kernel in combination with another kernel. For more information on kernel combinators, see the **Combining kernels** section."
   ]

    "\n",
    "The Harmonic Oscillator kernel is a stationary periodic kernel. It has the following definition:\n",
    "\n",
    "$k(x, x^{\\prime}) = \\sigma^2 cos\\left(\\frac{2\\pi}{p} \\cdot (x-x^{\\prime})\\right)$\n",
    "\n",
    "It has two parameters, the variance $\\sigma^2$, and the cycle period $p$ in radius unit."

    "from markovflow.kernels.piecewise_stationary import PiecewiseKernel\n",

    "\n",
    "\n",

    "\n",

    "    for l in range(num_inducing + 1)\n",

    "\n",
    "[k.set_state_mean(state_means[i]) for i, k in enumerate(ks)]\n",
    "\n",
    "kernel = PiecewiseKernel(\n",
    "    ks, tf.convert_to_tensor(change_points, dtype=default_float()))\n",
    "\n",

    "plt.plot(X, Ys)\n",
    "plt.hlines(state_means[:, 0], xmin=X.min(), xmax=X.max(), ls='--', color='k', alpha=.3, label='offset')\n",
