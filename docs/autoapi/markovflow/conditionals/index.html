\[p(x‚Çú) = ‚à´ d[x‚Çã, x‚Çä] p(x‚Çú|x‚Çã, x‚Çä) q(x‚Çã, x‚Çä; m‚Çú, S‚Çú) = ùìù(P‚Çú m‚Çú, T‚Çú + P‚Çú S‚Çú P‚Çú·µÄ)\]

\[p(x‚Çú|[x‚Çã, x‚Çä] = m‚Çú) = ùìù(P‚Çú @ [x‚Çã, x‚Çä], T‚Çú)\]

Note

  • \(p\) is the density over state trajectories specified by the kernel

    \[\begin{split}x‚Çä = arg min‚Çì \{|x-x‚Çú|, x ‚àà \verb|training_time_point and |x>x‚Çú\}\\ x‚Çã = arg min‚Çì \{|x-x‚Çú|, x ‚àà \verb|training_time_point and |x‚©Ωx‚Çú\}\end{split}\]

Compute \(F‚Çú, G‚Çú, L‚Çú\). Such that:

\[p(x·µâ‚Çú | x·∂ú‚Çú‚Çã‚ÇÅ, x·∂ú‚Çú‚Çä‚ÇÅ) = ùìù(x·µâ‚Çú; F‚Çú @ x·∂ú‚Çú‚Çã‚ÇÅ + G‚Çú @ x·∂ú‚Çú‚Çä‚ÇÅ, T‚Çú = (L‚Çú L‚Çú·µÄ)‚Åª¬π = L‚Çú‚Åª·µÄL‚Çú‚Åª¬π)\]

‚Ä¶where superscripts \(e\) and \(c\) refer to explained and conditioning respectively.

Note

\(x·µâ\) and \(x·∂ú\) must be sorted, such that:

\[x·µâ‚ÇÄ < x·∂ú‚ÇÄ < x·µâ‚ÇÅ < ... < x·µâ‚Çú < x·∂ú‚Çú < x·∂ú‚Çú‚Çä‚ÇÅ < x·∂ú‚Çú‚Çä‚ÇÅ < ...\]

‚Ä¶where \(len(x·µâ) = len(x·∂ú)\) or \(len(x·µâ) = len(x·∂ú) + 1\).

     |  G‚ÇÅ·µÄ, F‚ÇÇ·µÄ           [      |]             |   L‚ÇÇ‚Åª·µÄ            |
     |     , G‚ÇÇ·µÄ,‚ã±         [      |]             |      L‚ÇÉ‚Åª·µÄ         |
     |           ‚ã± ‚ã±       [      |]             |        ‚ã±          |
     |             ‚ã± F‚Çô‚Çã‚ÇÅ·µÄ [      |]             |          ‚ã±        |
     |               G‚Çô‚Çã‚ÇÅ·µÄ [ F‚Çô·µÄ  |]             |            L‚Çô‚Åª·µÄ   |

\(L\) is the Cholesky factor of the conditional precision \(x·µâ|x·∂ú\).

Statistics \(F\) and \(G\) are computed from the conditional mean projection parameters \(D\) and \(E\) defined by \(ùîº x·µâ‚Çô|x·∂ú = D‚Çô @ x·∂ú‚Çô‚Çã‚ÇÅ + E‚Çô @ x·∂ú‚Çô\).

Solving the system \(- (L‚Åª·µÄ U·µÄ x·∂ú)‚Çô = D‚Çô @ x·∂ú‚Çô‚Çã‚ÇÅ + E‚Çô @ x·∂ú‚Çô\), we get \(G‚Çô‚Çã‚ÇÅ·µÄ = -L‚Çô·µÄ D‚Çô\) and \(F‚Çô·µÄ = -L‚Çô·µÄ E‚Çô\).

-| L‚ÇÅ‚Åª·µÄF‚ÇÅ·µÄ x·∂ú‚ÇÅ                     | = | E‚ÇÅ x·∂ú‚ÇÅ

Remarks on size:

  • When splitting \(x\) of size \(n\) into odd and even, you get \(n·µâ = (n+1)//2\) and \(n·∂ú = n//2\)

  • At each level, cyclic reduction statistics have shape:

    \[\begin{split}&- F : n·∂ú\\ &- G : n·µâ - 1\\ &- L : n·µâ\\\end{split}\]

  • \(F‚ÇÄ\) is not defined (there is no time point below \(x·µâ‚ÇÄ\))

  • The last element \(G\) may not be defined if \(len(x·µâ) = len(x·∂ú)\)
