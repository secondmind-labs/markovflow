A base class for likelihoods, which specifies an observation model connecting the latent functions (‚ÄòF‚Äô) to the data (‚ÄòY‚Äô).

All of the members of this class are expected to obey some shape conventions, as specified by latent_dim and observation_dim.

If we‚Äôre operating on an array of function values ‚ÄòF‚Äô, then the last dimension represents multiple functions (preceding dimensions could represent different data points, or different random samples, for example). Similarly, the last dimension of Y represents a single data point. We check that the dimensions are as this object expects.

The return shapes of all functions in this class is the broadcasted shape of the arguments, excluding the last dimension of each argument.
